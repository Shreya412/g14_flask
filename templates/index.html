This is an analytical experiment to analyze the performance of inference techniques. Each experiment is a function of five parameters, representing cross sections, and the output is a single scalar representing the eigenvalue.

All cross-section parameter variations are assumed to be deviations from reference values, with the covariance structure here.

Upon selection, each of the four experiments will reveal a corresponding eigenvalue measurement along with its uncertainty.

The measurements are calculated at the true value for the cross sections, which isn't provided with this benchmark. The true value is selected to be statistically consistent with the prior.
